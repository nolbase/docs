# Audiences

> Build, segment, and target your ideal audience.

**Module slug:** `audiences`
**Category:** Testing & Growth
**URL:** `/platform/audiences`

## Overview

Audiences is NolBase's segmentation and targeting engine. Build dynamic audience segments based on behavior, demographics, and engagement data — then sync those segments directly to ad platforms for precision targeting.

Stop targeting broad audiences and hoping for the best. Audiences lets you build segments that match your ideal customers, keep them updated automatically, and push them to every channel where you run campaigns.

## Key Features

### Smart Segmentation Rules
Build segments with flexible logic:
- AND/OR conditions across any data point
- Demographics (age, location, company size, industry)
- Behavior (page visits, email opens, purchases, feature usage)
- Engagement (last activity, frequency, recency)
- Custom properties (tags, scores, lifecycle stage)
- Nested conditions for complex targeting

### Behavioral Targeting
Target based on what people do, not just who they are:
- Page visit tracking
- Email engagement (opens, clicks, replies)
- Purchase history and order value
- Feature usage patterns
- Content consumption behavior

### Lookalike Audience Builder
Find more people like your best customers:
- Build lookalike models from high-value segments
- Configurable similarity thresholds
- Platform-specific lookalike generation (Meta, Google, LinkedIn)

### List Management & Hygiene
Keep your data clean automatically:
- Import via CSV, Excel, or API
- Automatic deduplication on import and ongoing
- Email validation (syntax, MX records, disposable domain detection)
- Bounce processing and unsubscribe handling
- Inactive contact flagging

### Ad Platform Sync
Push segments directly to ad platforms:
- **Meta Ads** — Custom Audiences sync
- **Google Ads** — Customer Match sync
- **LinkedIn Ads** — Matched Audiences sync
- One-click sync with automatic updates
- Real-time segment refresh as data changes

### Audience Analytics & Insights
Understand your segments:
- Demographic breakdowns and distributions
- Engagement trends over time
- Growth and churn rates
- Overlap analysis between segments (avoid over-messaging)
- Segment performance comparison

## How It Works

1. **Define** — Create segments using the visual rule builder
2. **Populate** — Segments populate automatically from your data
3. **Analyze** — Review segment size, composition, and engagement patterns
4. **Sync** — Push segments to ad platforms with one click
5. **Evolve** — Segments update dynamically as new data comes in

## Platform Integration

- **Publish** → Target specific segments with each campaign deployment
- **Analytics** → Measure performance by audience segment
- **A/B Tests** → Test messaging across different audience segments
- **Automate** → Trigger workflows when contacts enter or exit segments
- **Email API** → Send targeted email campaigns to specific segments
