# Automate

> Eliminate repetitive marketing tasks with powerful automation.

**Module slug:** `automate`
**Category:** Operations & Delivery
**URL:** `/platform/automate`

## Overview

Automate lets you build trigger-based workflows that connect actions across every NolBase module. Content approval pipelines, campaign launch sequences, performance alerts, and reporting workflows — all configured visually, without writing code.

If your team does it repeatedly, Automate can do it automatically.

## Key Features

### Visual Workflow Builder
Build automation with a drag-and-drop canvas:
- No coding required
- Visual representation of every step
- Real-time preview of workflow logic
- Clone and modify existing workflows

### Trigger-Based Automation
Start workflows from any event in NolBase:
- **Schedule triggers** — Run at specific times or intervals
- **Event triggers** — React to module events (new content created, approval completed, performance threshold hit)
- **Webhook triggers** — Accept external events via HTTP
- **Manual triggers** — Run on-demand with one click

### Multi-Step Sequences
Chain actions with complex logic:
- Sequential and parallel execution paths
- Time delays and wait conditions
- Retry logic for failed steps
- Timeout handling

### Conditional Logic
Route workflows based on data:
- If/else branching
- Data filters and transformations
- Attribute-based routing (content type, campaign, audience)
- Switch/case for multiple conditions

### Template Library
Start with pre-built workflows for common patterns:
- Content approval pipeline
- Campaign launch checklist
- Weekly performance report
- Overspend alert
- Competitor activity notification

### Cross-Module Actions
Trigger actions in any NolBase module:
- Create content in Content Studio
- Submit for review in Approvals
- Schedule in Publish
- Send notification via Slack
- Update budget in Budgets
- Generate report in Analytics

## Execution & Debugging

- Detailed execution logs showing every step with timestamps
- Step-by-step results inspection
- Replay failed executions
- Test individual actions in isolation
- Error notifications with context

## How It Works

1. **Trigger** — Define what starts the workflow (event, schedule, webhook, or manual)
2. **Build** — Add steps, conditions, and actions using the visual builder
3. **Test** — Run the workflow in test mode to verify behavior
4. **Activate** — Enable the workflow for live execution
5. **Monitor** — Track executions, debug failures, and optimize

## Platform Integration

Automate connects to every NolBase module:
- **Content Studio** → Auto-generate content based on triggers
- **Approvals** → Route content for review automatically
- **Publish** → Schedule deployments based on approval events
- **Analytics** → Generate reports on schedule
- **Budgets** → Alert on overspend thresholds
- **Intelligence** → React to competitive intelligence signals
- **Slack** → Send team notifications at any workflow step
- **Zapier** → Connect to 5,000+ external apps
