# Export

> Package creatives in platform-ready formats.

**Module slug:** `export`
**Category:** Content & Creative

## Overview

Export handles the last mile between content creation and deployment. It packages your marketing creatives in platform-specific dimensions and formats — so your Meta ads, Google display banners, LinkedIn images, and TikTok videos are always spec-compliant without manual resizing.

## Key Features

### Platform-Specific Export
Export creatives formatted for each ad platform's requirements:
- Meta Ads dimensions (feed, stories, reels, carousel)
- Google Ads formats (responsive display, YouTube)
- LinkedIn Ads specifications
- TikTok Ads formats
- Social media post dimensions per platform

### Batch Processing
Export multiple creatives at once:
- Select multiple assets and export all at once
- Automated resizing and cropping for each format
- Compression optimization for each platform's file size requirements
- ZIP packaging for bulk downloads

### Naming Conventions
Keep exported files organized:
- Configurable naming templates (campaign_platform_size_variant)
- Automated file naming based on asset metadata
- Folder structure by campaign or platform

### Format Conversion
Convert between formats automatically:
- Image format conversion (PNG, JPG, WebP, SVG)
- Video format conversion and compression
- Resolution scaling with quality preservation

## How It Works

1. **Select** — Choose creatives from Content Studio or asset library
2. **Configure** — Select target platforms and formats
3. **Export** — Batch export with automatic resizing and format conversion
4. **Download** — Direct download or push to cloud storage

## Platform Integration

- **Content Studio** → Export AI-generated creatives in any format
- **Publish** → Deployed campaigns use Export for format compliance
- **Approvals** → Export approved assets with audit metadata
