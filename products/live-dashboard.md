# Live Dashboard

> Real-time performance analytics across every channel.

**Module slug:** `pulse`
**Category:** Analytics & Intelligence
**URL:** `/platform/pulse`

## Overview

Live Dashboard provides real-time marketing performance data across all your advertising and social channels. While Analytics focuses on historical reporting and attribution, Live Dashboard gives you the "right now" view — what's performing, what's stalling, and what needs attention this moment.

Think of it as your marketing command center. One screen, every channel, updated continuously.

## Key Features

### Unified Real-Time Dashboard
See everything at a glance:
- Meta Ads, Google Ads, LinkedIn Ads, TikTok Ads
- Email campaign metrics
- Social media engagement
- Website traffic and conversions
- All updated in near real-time

### AI-Powered Insights
Don't just see data — understand it:
- Anomaly detection (sudden drops or spikes)
- Performance trend identification
- Optimization suggestions (pause underperforming ads, increase budget on winners)
- Cross-channel pattern recognition

### Custom Reporting
Build the views that matter to your team:
- Drag-and-drop widget builder
- Scheduled report delivery (email, Slack)
- PDF and CSV export
- White-label options for agency dashboards

### Spend Analysis
Real-time budget consumption monitoring:
- Spend pacing across channels
- Cost-per-result trends
- Budget utilization rates
- Overspend warnings

### Campaign Comparison
Compare performance across campaigns:
- Side-by-side campaign comparison
- Channel-level performance breakdowns
- Time period comparisons (this week vs. last week)
- Benchmark tracking against goals

## Data Refresh Rates

| Source | Refresh Interval |

## How It Works

1. **Connect** — Link your ad accounts and data sources
2. **Configure** — Set up your dashboard layout with relevant widgets
3. **Monitor** — Watch real-time performance data flow in
4. **Act** — Respond to AI insights and anomaly alerts immediately
5. **Share** — Export or schedule dashboards for stakeholders

## Platform Integration

- **Analytics** → Live Dashboard feeds data into historical Analytics for trending
- **Budgets** → Real-time spend data correlates with budget tracking
- **Publish** → See performance of deployed campaigns immediately
- **Automate** → Trigger workflows based on real-time performance thresholds

## Who It's For

- **Performance marketers** monitoring live campaign results
- **Marketing directors** needing a daily command-center view
- **Agency teams** tracking multiple client accounts in real time
